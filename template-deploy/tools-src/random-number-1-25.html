What is a random number between 1 and 25?

The 1–25 range is a natural fit for groups of up to 25 people, quarter-century milestones, and games using a 25-space board or card deck. Each of the 25 integers has a 4% probability per generate. The tool uses JavaScript's Math.random(), seeded from hardware entropy, for statistically uniform results.

Pool mode: five groups of five

Pool mode divides 25 numbers into a shuffled queue and draws them without replacement. This is ideal for assigning unique IDs in groups of up to 25, calling on students in a class of 25, or running a lottery with exactly 25 tickets. The distribution chart shows results in five buckets of five (1–5, 6–10, 11–15, 16–20, 21–25) for easy visual verification.

Common uses for 1–25

Twenty-five is a natural boundary: 5×5 grids, a quarter of 100, class sizes, bingo-style games with 25 spaces, and competition brackets of 25 participants. Pool mode ensures every number in the range is used exactly once before repeating.

Frequently asked questions

Is this truly random?
Yes. Math.random() seeded from hardware entropy. Each of the 25 integers has a 4% (1-in-25) probability per generate — statistically uniform.
What is pool mode for 1–25?
Pool mode draws all 25 numbers in shuffled order without replacement. Every number appears once before any repeat. The progress bar tracks how many remain in the current cycle.
Can I exclude specific numbers?
Yes. Click "Exclude numbers" and enter a comma-separated list. Excluded numbers are skipped in both normal and pool modes.
