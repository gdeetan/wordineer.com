How to use the wheel of names

Type or paste your list of names into the text box — one name per line, or separated by commas. You can paste directly from a Google Sheets column, a class roster, or any plain-text list. The wheel updates live as you type, so you'll see each name appear as a colored segment immediately.

Once your names are loaded, press Spin the wheel (or click the wheel itself, or press Space or Enter). The wheel spins for 3–4 seconds and slows to a stop. A winner card pops up with the selected name. From there you can spin again, or remove that name and spin again — useful for elimination-style games.

Round robin mode

Switch to Round Robin mode using the toggle above the name list. In this mode the wheel tracks which names have already been picked in the current round. Each name can only win once per round — once everyone has been selected, the pool resets automatically and a new round begins.

Teachers love round robin mode for fair classroom participation: every student gets called on before anyone is repeated. A status line below the toggle shows how many students have been picked and how many remain this round, so you always know where you are.

Shareable URL

As soon as you add names, a shareable URL appears in the box below the name list. This URL encodes your full name list as a query parameter — no server, no account, no saved data. Bookmark it or share it with a colleague and the same list loads instantly when the link is opened.

This makes it easy to set up your wheel at home and pull it up on a classroom projector without retyping anything. The URL updates automatically whenever you add or remove names.

Classroom uses

The wheel of names is designed to be projector-friendly. The colorful spinning wheel is easy to see from the back of a classroom, and the large winner card is readable at a distance. No student accounts are needed — the tool runs entirely in the browser.

Common classroom uses include random student selection for questions or presentations, assigning groups by spinning twice per student, picking reading partners, and determining the order of team presentations. The tool works on any device — phone, tablet, laptop, or desktop — with no installation required.

Frequently asked questions

How do I add names to the wheel?
Type or paste names into the text box — one name per line, or separated by commas. The wheel updates instantly as you type. You can paste a list directly from a spreadsheet, Google Sheets, or any text editor and the names will appear on the wheel immediately.
What is round robin mode?
Round robin mode ensures every name is picked exactly once before any name repeats. Once the last name is drawn, the pool resets automatically and a new round begins. It's ideal for classroom participation, fair turn order, or any situation where equal selection matters.
Can I save my name list?
Yes. A shareable URL is generated automatically whenever you add names. Copy that link and bookmark it — reopening the URL loads your exact name list without needing an account or sign-in. You can also share the link with a colleague so they get the same list.
How many names can the wheel hold?
There is no hard limit built into the tool. Practically, the wheel stays readable up to about 30–40 names. Beyond that the segments become too narrow to show labels clearly, though the spinner still works correctly and picks a random name with equal probability for each entry.
Is this free to use?
Yes, completely free. No account, no sign-up, no download required. Open the page, paste your names, and spin. The tool runs entirely in your browser and does not send any data to a server.
