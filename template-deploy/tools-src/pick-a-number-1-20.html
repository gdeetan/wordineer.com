How to play "I'm thinking of a number between 1 and 20"

The classic number guessing game: one player picks a secret number between 1 and 20, the other player(s) try to guess it. After each wrong guess, the picker gives a "higher" or "lower" hint. The player who guesses correctly wins. This tool does the picking for you — eliminating any temptation to change the number mid-game. Just generate, keep the screen hidden, and start taking guesses.

Blind pick mode: simultaneous reveal

Blind pick mode offers a different game structure: each player secretly picks a number between 1 and 20, then both are revealed at the same time. The higher number wins. This is perfect for tiebreakers, choosing turn order, or any two-player decision where simultaneous commitment prevents gamesmanship.

Fair decisions and classroom uses

Beyond games, the 1–20 picker is a classic tool for fair decisions: who goes first, who picks the movie, who does the task nobody wants. Teachers use pool mode to assign each student a unique "secret number" for activities. The exclude feature lets you narrow the range — for example, pick from 1–15 for a class of 15 by excluding 16–20.

Frequently asked questions

How do I use this for a guessing game?
Click "Pick a number" to generate a secret number between 1 and 20. Keep the screen turned away or hidden. Let the other player(s) guess, giving "higher" or "lower" hints until they get it. The picker can't cheat — the number is locked in as soon as it's generated.
Is the number truly random?
Yes. The tool uses Math.random() seeded from hardware entropy. Each value from 1 to 20 has an equal 5% chance on every pick — no pattern, no bias.
What is blind pick mode?
Both players secretly pick a number between 1 and 20, then reveal simultaneously. The higher number wins. Useful for tiebreakers where neither player should see the other's choice first.
Can I use pool mode for a guessing game?
Yes — pool mode cycles through all 20 numbers in shuffled order, so no two rounds use the same number. This is great for running multiple rounds of the guessing game in a classroom, ensuring each round has a unique secret number.
