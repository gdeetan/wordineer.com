What is lorem ipsum?

Lorem ipsum text traces back to Cicero's philosophical work de Finibus Bonorum et Malorum, written around 45 BC. A typesetter in the 1500s scrambled a passage from that text — starting with "Lorem ipsum dolor sit amet" — to create filler copy that looks like Latin but doesn't carry any meaning. Because it's meaningless, it lets reviewers focus on layout and typography rather than being distracted by real content.

When to use placeholder text — and when not to

Lorem ipsum is the right call when you're testing font choices, spacing, or grid structure and real content isn't available yet. It's the wrong call when you're testing whether copy fits, evaluating how readers will scan the page, or showing mockups to stakeholders who'll fixate on the fake Latin. For those cases, switch to Readable English mode to get real sentences that still function as filler.

Dev tip: built-in lorem ipsum in your editor

If you use VS Code or JetBrains IDEs, you already have lorem ipsum built in. Type lorem in an HTML file and hit Tab — Emmet expands it to a paragraph automatically. This generator is useful when you need a specific word count, HTML output, multiple formats, or an alternative to the classic Latin.

Frequently asked questions

What is lorem ipsum?
Lorem ipsum is placeholder text derived from Cicero's "de Finibus" (45 BC), scrambled so it looks like Latin but carries no meaning. It's used in design and publishing to fill space before real content is ready.
How do I generate 3 paragraphs of lorem ipsum?
Set the unit to "Paragraphs", enter 3 in the count field, and click Generate. Copy as plain text, HTML, or Markdown using the format tabs.
Can I get lorem ipsum as HTML?
Yes — switch to the HTML tab to get output wrapped in <p> tags (or <ul><li> for list items), ready to paste directly into markup.
What is Word Soup mode?
Word Soup pulls random real English words of varied lengths to create filler that tests how your typography handles a mix of short and long words — useful for stress-testing line breaks and kerning.
What URL parameters does this generator support?
You can bookmark a specific configuration: ?unit=paragraphs&count=3&format=html&mode=latin. The page restores that state on load — useful for devs who want a repeatable setup.
