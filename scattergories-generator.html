    How to use the Scattergories generator

    Scattergories is a word game where every answer must start with a specific letter and fit a given category — and you only score if nobody else picked the same answer.

    1. Roll a letter — click the Roll button to get a random letter from the official 20-letter pool (Q, U, V, X, Y, and Z are excluded because very few common answers start with them).
    2. Get your categories — 12 categories appear on the right. Click "New list" to swap to a different set at any time.
    3. Start the timer — 3 minutes is the standard round length. Everyone writes down one answer per category that starts with the rolled letter.
    4. Score — when time's up, players read their answers aloud. Any answer that matches another player's answer scores zero. Unique answers score one point. Alliterative answers (both words start with the letter) score two.

    Playing online or remotely

    Share your screen on Zoom, Discord, or a TV, and everyone plays from one shared view. The generator handles the letter roll, category shuffle, and timer so you can focus on playing.

    Frequently asked questions

    What letters are excluded and why?

    The six letters Q, U, V, X, Y, and Z are excluded by default because very few everyday words start with them. Toggle "Use all 26 letters" if you want the harder version.

    How many categories are in a Scattergories round?

    A standard round uses 12 categories. Click "New list" to get a fresh set of 12 from the same theme.

    Can I use this for remote play?

    Yes — share your screen so everyone sees the same letter and categories. When time's up, read answers aloud on your call.

    How do I print a Scattergories card?

    Click "Print card" for a clean layout showing the letter and 12 numbered blank lines.

    What does the category theme filter do?

    It draws categories from a specific pool — Classic, Kids, Adults, Holiday, Food, Pop culture, or Hard. "All themes" mixes them together.

    What is Play Mode?

    Play Mode expands the tool to fill the screen with a bigger letter and categories — optimized for TV or group viewing.

    Who uses the Scattergories generator

    • Game night hosts who want a no-setup alternative to finding the physical game
    • Teachers using Scattergories as a vocabulary and quick-thinking exercise
    • Remote teams running virtual happy hours and ice-breaker games
    • Families looking for a screen-time activity everyone can play
